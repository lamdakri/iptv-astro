# Playlist M3U para IPTV 2026

A playlist M3U é o coração de todo sistema IPTV. Ela contém todos os endereços de canais, categorias e metadados para sua experiência IPTV.

## O que é uma playlist M3U?

M3U significa **MP3 URL**, um formato de texto simples originalmente criado para playlists de música. Para IPTV, é usado para armazenar endereços, nomes e categorias de canais de TV.

## URL M3U vs Arquivo M3U

- **URL M3U**: Link web para playlist online. Ideal porque atualiza automaticamente.
- **Arquivo M3U**: Arquivo .m3u local no dispositivo. Acesso offline mas requer atualização manual.

## Como carregar uma playlist M3U

### No Android (TiviMate)
1. Abra o TiviMate
2. **Adicionar Playlist** > **Playlist M3U**
3. Cole a URL e clique **Avançar**

### No Firestick (IPTV Smarters Pro)
1. Abra o IPTV Smarters Pro
2. **Carregar por URL**
3. Cole a URL M3U e clique **Carregar**

## Problemas comuns M3U

| Problema | Solução |
|----------|----------|
| Playlist não carrega | Verifique conexão e URL |
| Nenhum canal visível | Limpe cache do app |
